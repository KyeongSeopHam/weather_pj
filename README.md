## 💡 날씨(Api)를 활용해서 CRUD 날씨,일기를 작성/조회/수정/삭제 Back-end 구현하기!!!

- 프로젝트 기간 : 2022.10.23 ~ 2022.10.29








## 💡 주요기능
1. 날씨 API 정보 가져오기 기능 구현 (https://openweathermap.org/api)
2. 날씨,일기를 작성 (C) , 날씨,일기를 조회(R), 수정(U), 삭제(D)
